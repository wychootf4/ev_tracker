隐私政策

EV助手 (EV Charge Tracker)

最后更新日期：2025年12月10日

简而言之：EV助手完全在本地运行，不收集、不上传、不共享您的任何个人数据。您的所有行程记录、充电信息和车辆数据都仅保存在您的设备上。

1. 概述

感谢您使用 EV助手（以下简称"本应用"）。我们非常重视您的隐私权。本隐私政策旨在帮助您了解我们如何处理您的信息。

2. 数据收集与存储

2.1 我们不收集的数据

本应用不会收集、传输或存储以下任何数据到远程服务器：

  • 个人身份信息（姓名、电话、邮箱等）
  • 车辆识别信息（车牌、VIN 码等）
  • 行程记录和充电历史
  • 地理位置数据
  • 照片或相册内容
  • 使用习惯和分析数据

2.2 本地数据存储

所有您输入的数据都使用 iOS 系统的 Core Data 框架加密存储在您的设备本地，包括：

  • 车辆信息（电池容量、总里程等）
  • 行程记录（起始里程、电量、续航等）
  • 充电记录（充电地点、费用、时间等）
  • 用户偏好设置

这些数据受到 iOS 系统级加密保护，仅您本人可以访问。我们无法访问这些数据。

3. 权限使用说明

本应用需要以下权限来提供核心功能，所有权限都仅在必要时使用，且数据不会离开您的设备：

3.1 📷 相机权限

  • 用途：拍摄车辆仪表盘或手机 App 截图，使用 OCR 技术自动识别里程、电量和续航数据
  • 处理方式：照片仅在设备本地进行文字识别，识别完成后不会保存照片
  • 数据去向：识别后的数据仅保存在本地数据库，照片不会上传或保存

3.2 📸 相册权限

  • 用途：从相册选择车辆运行数据的截图进行 OCR 识别
  • 处理方式：仅读取您选择的照片进行文字识别，不会访问其他照片
  • 数据去向：识别后的数据仅保存在本地数据库，不会上传照片

3.3 📍 位置权限（仅在使用时）

  • 用途：
    • 自动记录充电地点的位置信息和地址
    • 获取当前位置的天气温度数据（通过 Open-Meteo 免费 API）
  • 处理方式：仅在您添加充电记录时获取一次位置，温度数据来自第三方天气服务
  • 数据去向：位置坐标和地址仅保存在本地数据库，不会上传到我们的服务器
  • 第三方服务：位置坐标会发送到 Open-Meteo API 以获取温度数据，该服务不收集个人信息（详见第 4 节）

4. 第三方服务

4.1 天气数据服务

本应用使用 Open-Meteo JMA API 获取天气温度数据：

  • 服务提供商：Open-Meteo（open-meteo.com）
  • 数据传输：仅发送地理坐标（经纬度）以获取当前温度
  • 隐私保护：该服务完全免费且不需要 API 密钥，不收集用户个人信息
  • 隐私政策：Open-Meteo Terms

4.2 Apple 系统服务

本应用使用以下 Apple 提供的系统服务：

  • Core Data：本地数据存储和加密
  • Vision Framework：设备端 OCR 文字识别（完全在本地运行）
  • Core Location：位置服务和地理编码
  • WidgetKit：桌面小组件显示
  • App Intents：Siri 快捷指令集成

这些服务均受 Apple 的隐私政策保护，详情请参阅 Apple 隐私政策。

5. 数据安全

我们采取以下措施保护您的数据安全：

  • 本地加密存储：使用 iOS Core Data 的加密特性保护数据
  • 沙盒隔离：应用数据存储在 iOS 沙盒中，其他应用无法访问
  • 无服务器架构：没有后端服务器，无法被黑客攻击或数据泄露
  • 代码安全：不使用第三方追踪 SDK 或分析工具

6. 数据共享与披露

由于本应用不收集任何数据，因此：

  • ✅ 我们不会向任何第三方出售、租赁或共享您的数据
  • ✅ 我们不会将数据用于广告或营销目的
  • ✅ 我们不会与数据分析公司共享数据
  • ✅ 我们无法响应政府或执法机关的数据请求（因为我们没有数据）

7. 数据导出与删除

7.1 数据导出

您可以随时通过应用内的"设置 → 导出数据"功能将所有数据导出为 CSV 文件，以便备份或迁移到其他工具。

7.2 数据删除

您可以随时：

  • 在应用内删除单条记录（滑动删除）
  • 删除整个应用以清除所有本地数据

由于数据仅存储在本地，删除应用后数据将被永久删除，无法恢复。

8. 儿童隐私

本应用不针对 13 岁以下儿童，也不会故意收集儿童的个人信息。如果您是家长或监护人，发现您的孩子向我们提供了个人信息，请联系我们，我们将及时删除相关信息。

9. 国际用户

本应用在全球范围内可用。由于所有数据都存储在您的设备上，不存在跨境数据传输的问题。唯一的例外是温度数据请求会发送到 Open-Meteo 的服务器（位于欧洲），但仅包含地理坐标，不包含个人信息。

10. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在此页面发布，并更新"最后更新日期"。重大变更会在应用内通知您。
